## Authentication

All API requests require an API Key in the request header:
```
Authorization: Bearer <YOUR_API_KEY>
```

Get your API key from the [Thunderbit Dashboard](https://thunderbit.com).

## Quick Start

Distill a web page into clean Markdown with a single API call:

```bash
curl -X POST https://openapi.thunderbit.com/openapi/v1/distill \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```

Extract structured data from any web page:

```bash
curl -X POST https://openapi.thunderbit.com/openapi/v1/extract \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/product", "schema": {"type": "object", "properties": {"name": {"type": "string"}, "price": {"type": "number"}}}}'
```
