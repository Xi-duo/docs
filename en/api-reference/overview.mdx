## Base URL

```
https://open.thunderbit.com/v1
```

## Authentication

All endpoints use HTTP Bearer (JWT):

```
Authorization: Bearer YOUR_API_KEY
```

## Error Responses

<Warning>
Errors return a consistent envelope: `{ "success": false, "error": { "code": "...", "message": "..." } }`
</Warning>

| Code                  | Meaning                             |
| --------------------- | ----------------------------------- |
| `INVALID_URL`         | Invalid URL format                  |
| `URL_NOT_ACCESSIBLE`  | Target URL unreachable              |
| `TIMEOUT`             | Request timeout                     |
| `QUOTA_EXCEEDED`      | Quota exhausted                     |
| `RATE_LIMITED`        | Rate limit triggered                |
| `INVALID_SCHEMA`      | Invalid JSON Schema                 |
| `EXTRACTION_FAILED`   | AI extraction failed                |
| `BATCH_SIZE_EXCEEDED` | Batch request exceeds maximum       |
| `INVALID_WEBHOOK_URL` | Webhook URL invalid or not HTTPS    |
