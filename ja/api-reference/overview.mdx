## Base URL

```
https://open.thunderbit.com/v1
```

## 認証

すべてのエンドポイントで HTTP Bearer (JWT) を使用：

```
Authorization: Bearer YOUR_API_KEY
```

## エラーレスポンス

<Warning>
エラーは共通フォーマットで返されます：`{ "success": false, "error": { "code": "...", "message": "..." } }`
</Warning>

| コード                | 意味                              |
| --------------------- | --------------------------------- |
| `INVALID_URL`         | URL フォーマットが不正            |
| `URL_NOT_ACCESSIBLE`  | 対象 URL にアクセスできない       |
| `TIMEOUT`             | リクエストタイムアウト            |
| `QUOTA_EXCEEDED`      | クォータ上限に到達                |
| `RATE_LIMITED`        | レート制限を超過                  |
| `INVALID_SCHEMA`      | JSON Schema が不正                |
| `EXTRACTION_FAILED`   | AI 抽出に失敗                     |
| `BATCH_SIZE_EXCEEDED` | バッチリクエスト数が上限を超過    |
| `INVALID_WEBHOOK_URL` | Webhook URL が無効または非 HTTPS  |
