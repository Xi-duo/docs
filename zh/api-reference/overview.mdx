## Base URL

```
https://open.thunderbit.com/v1
```

## 认证

所有端点使用 HTTP Bearer（JWT）：

```
Authorization: Bearer YOUR_API_KEY
```

## 错误响应

<Warning>
错误使用统一信封：`{ "success": false, "error": { "code": "...", "message": "..." } }`
</Warning>

| 错误码                | 含义                        |
| --------------------- | --------------------------- |
| `INVALID_URL`         | URL 格式无效                |
| `URL_NOT_ACCESSIBLE`  | 目标 URL 不可访问           |
| `TIMEOUT`             | 请求超时                    |
| `QUOTA_EXCEEDED`      | 配额耗尽                    |
| `RATE_LIMITED`        | 触发速率限制                |
| `INVALID_SCHEMA`      | JSON Schema 无效            |
| `EXTRACTION_FAILED`   | AI 提取失败                 |
| `BATCH_SIZE_EXCEEDED` | 批量请求超过上限            |
| `INVALID_WEBHOOK_URL` | Webhook URL 无效或非 HTTPS  |
