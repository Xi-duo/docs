## 核心能力

<CardGroup cols={2}>
  <Card title="网页蒸馏" icon="wand-magic-sparkles">
    把网页转成干净的 Markdown 格式，完美适配 RAG 和 AI 应用。
  </Card>
  <Card title="AI 结构化提取" icon="brain">
    使用 JSON Schema 或自然语言提示提取结构化数据。
  </Card>
  <Card title="批量处理" icon="layer-group">
    异步批量处理多个 URL，支持任务管理和 Webhook 回调。
  </Card>
  <Card title="企业级" icon="shield-halved">
    自动处理 JS 渲染、反爬、代理和动态内容。
  </Card>
</CardGroup>

## 认证

<Note>
所有请求需要 Bearer Token。从 [Thunderbit Dashboard](https://thunderbit.com) 获取你的 API Key。
</Note>

```bash
Authorization: Bearer YOUR_API_KEY
```
