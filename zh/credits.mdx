---
title: "额度"
description: "不同操作的 API 额度消耗"
---

Each API call consumes credits from your account:

| Operation | Credits per Request |
|-----------|-------------------|
| Distill (single page) | 1 credit |
| Extract (single page) | 20 credits |
| Batch Distill | 1 credit per URL |
| Batch Extract | 20 credits per URL |
| Batch Status Check | Free |

Credits are deducted from bonus credits first, then monthly plan credits.
