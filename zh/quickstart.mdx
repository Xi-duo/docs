---

## 1. 获取 API Key

<Steps>
  <Step title="注册账号">
    在 [thunderbit.com](https://thunderbit.com) 注册。
  </Step>
  <Step title="复制 Key">
    **Dashboard → API Keys** 里找到你的 Key。
  </Step>
  <Step title="设置环境变量">
    ```bash
    export THUNDERBIT_API_KEY="sk_live_..."
    ```
  </Step>
</Steps>

## 2. 蒸馏一个页面

```bash
curl 'https://open.thunderbit.com/v1/distill' \
  -H "Authorization: Bearer $THUNDERBIT_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{"url": "https://example.com/article"}'
```

## 3. 提取结构化数据

```bash
curl 'https://open.thunderbit.com/v1/extract' \
  -H "Authorization: Bearer $THUNDERBIT_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{
    "url": "https://example.com/product",
    "schema": {
      "type": "object",
      "properties": {
        "title": {"type": "string"},
        "price": {"type": "number"}
      }
    }
  }'
```

<Check>
响应应该包含 `success: true`。接下来探索完整的 [API 参考](/zh/api-reference/overview)。
</Check>
